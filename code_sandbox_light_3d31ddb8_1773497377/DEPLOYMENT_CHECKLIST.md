## Deployment Instructions

### Option 1: Publish Tab (Recommended)
1. Go to the **Publish** tab in your interface
2. Click **Publish Project**
3. Wait for deployment to complete
4. Copy the live URL provided
5. Share your Pranayama app with the world! 🎉

### Option 2: Manual Deployment
If deploying elsewhere:

1. **GitHub Pages**
   ```bash
   git init
   git add .
   git commit -m "Initial pranayama app"
   git branch -M main
   git remote add origin [your-repo-url]
   git push -u origin main
   # Enable GitHub Pages in repo settings
   ```

2. **Netlify Drop**
   - Drag project folder to netlify.com/drop
   - Instant deployment
   - Get shareable URL

3. **Vercel**
   ```bash
   vercel deploy
   ```

4. **Any Static Host**
   - Upload all files maintaining folder structure
   - Ensure index.html is at root level
   - CSS and JS paths are relative
